자세히 보기

PocaskE&M: 실시간 광고 플랫폼 안정성 확보

퍼포먼스 마케팅 광고 솔루션 '포인트 크러쉬'의 기능 개발 및 유지보수를 담당했습니다.

주요 담당 분야

  • Express 기반 리워드 앱 서버 개발 및 운영
  • 기존 Kafka 기반 크롤링 인프라 유지보수 및 고도화
  • Spring Boot 기반 배치 서버 개발
  • 실시간 통계 시스템 데이터 파이프라인 설계

사용 기술

Express NestJS Spring Boot JPA Kafka Redis

도전 과제

  • DB 데드락: WAS 6대, 각 8개 프로세스(총 48개 클러스터)에서 발생하는 동시 요청으로 인한 DB 데드락
  • 인프라 한계: 클라우드 환경의 제약으로 인한 크롤링 속도 저하 및 잦은 IP 차단 문제

해결 방안

  • 이벤트 큐 도입: Redis를 활용하여 쓰기 작업을 비동기 벌크 처리 방식으로 전환, DB 부하 분산 및 데드락 해결
  • 온프레미스 인프라 구축: Tor 기반 IP 우회 로직을 포함한 크롤링 인프라를 직접 구축하여 데이터 수집 안정성 확보

기술 공유: Apache Kafka 세미나

안정적인 대규모 데이터 처리를 위해 도입한 Kafka는 저희 서비스의 핵심 기술 중 하나입니다. 이 기술에 대한 팀원들의 이해를 높이고 현재 적용된 아키텍처를 공유하기 위해 내부 기술 세미나를 진행했습니다.

그로잉랩: 금융 데이터 플랫폼 성능 최적화

개인 투자자를 위한 데이터 분석 플랫폼 '버틀러'의 백엔드, 관리자 페이지, 모바일 앱 개발을 담당했습니다.

주요 담당 분야

  • Express/Hapi 레거시 백엔드를 NestJS(TypeORM)로 마이그레이션
  • Github Packages를 활용한 공통 TypeORM 모델 라이브러리 개발
  • React 기반 관리자 페이지 기능 개발
  • React Native 기반 하이브리드 앱(iOS/Android) 개발

사용 기술

NestJS TypeORM React React Native MySQL

도전 과제

  • 느린 로딩 속도: S3에서 대용량 모놀리식 JSON 파일을 로드하는 구조로 인한 3초 이상의 응답 시간
  • 유지보수성 저하: Express, Hapi로 분산된 레거시 백엔드 구조

해결 방안

  • API 분리: 모놀리식 JSON 구조를 RDB 기반의 도메인별 API로 분리하여 로딩 속도를 100ms대로 개선
  • 백엔드 통합: 분산된 백엔드를 NestJS로 통합 마이그레이션하여 개발 생산성 및 유지보수성 향상
  • 코드 재사용: Github Packages를 도입하여 공통 TypeORM 모델을 라이브러리화하고 코드 중복 제거

Concent: 소셜 미디어 앱 인프라 비용 절감

말풍선 기반 소셜미디어 앱 'PerMirth'의 백엔드와 React Native 클라이언트 개발을 담당했습니다.

주요 담당 분야

  • HLS 스트리밍 도입을 통한 S3 비용 최적화
  • React Native 기반 앱 클라이언트 개발
  • CodePush를 활용한 앱 실시간 업데이트 적용
  • FFMPEG를 이용한 모바일 영상 인코딩 기능 구현

사용 기술

Spring Boot JPA React Native MongoDB AWS S3 CodePush

도전 과제

원본 영상 업로드 방식으로 인해 월간 청구되는 S3 비용 60만원은 스타트업에게 상당한 재정적 부담이었습니다.

해결 방안

HLS(HTTP Live Streaming) 기반 스트리밍 구조를 도입했습니다. 원본 영상을 업로드하는 대신, 영상을 .m3u8 플레이리스트와 .ts 세그먼트 파일로 분할하여 업로드하도록 전환하여 스토리지 사용량과 트래픽 비용을 최적화했습니다.

브릭메이트: 롯데홈쇼핑 플랫폼 고도화

롯데홈쇼핑에 고객사 파견 근무하며 대규모 레거시 커머스 플랫폼의 유지보수 및 기능 개발을 담당했습니다.

주요 경험

  • 대규모 트래픽 환경에서의 장애 대응 및 이슈 관리 경험
